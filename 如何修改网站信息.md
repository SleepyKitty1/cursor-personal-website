# 如何修改网站上的信息

所有内容都在 **`index.html`** 里用文字直接写好的，用任意文本编辑器（如记事本、VS Code、Cursor）打开该文件，按下面位置查找并修改即可。改完保存，刷新浏览器即可看到效果。

---

## 1. 首页（个人信息）

| 要改的内容 | 在 index.html 里搜什么 | 说明 |
|------------|------------------------|------|
| 导航栏左侧名字 | `logo-text` | 把「个人主页」改成你的名字或站点名。 |
| 首页大标题「你好，我是」 | `hero-greeting` | 可改成英文 "Hello, I'm" 或保留中文。 |
| **你的名字**（中间发光的那个） | `hero-name` | 改成你的真实姓名。 |
| 首页下方一段话（个人简介） | `hero-desc` | 在 `<p class="hero-desc">` 和 `</p>` 之间改文字。 |
| 「联系我」按钮跳转 | `href="#contact"` | 已指向本页的联系区块，一般不用改。 |
| 「查看简历」按钮 | `查看简历` 和 `href="#"` | 把 `href="#"` 改成你的简历链接；把「查看简历」改成你想要的文字。 |
| 首页下方社交图标链接 | `hero-social` 下面的两个 `<a>` | 第一个一般是 LinkedIn：把 `href="#"` 改成你的 LinkedIn 主页；第二个是邮箱：把 `mailto:your@email.com` 改成你的邮箱。 |

---

## 2. 关于我

| 要改的内容 | 在 index.html 里搜什么 | 说明 |
|------------|------------------------|------|
| 四个数字/指标 | `metric-value` 和 `metric-label` | 每个 `metric-card` 里有两个 `<span>`：上面是数字或关键词（如 8+、AI 方案），下面是说明（如「年经验」「产品策略」）。按你的经历改数字和文字即可。 |
| 「关于我」下面的两段话 | `about-text` | 在 `<div class="about-text">` 里的两个 `<p>...</p>` 中修改或重写。 |

---

## 3. 技能（我能带来）

| 要改的内容 | 在 index.html 里搜什么 | 说明 |
|------------|------------------------|------|
| 标题「我能带来：」 | `我能带来` | 直接改这几个字。 |
| 每条技能名称 | `skill-name` | 有 4 个 `skill-card`，每个里有一个 `<span class="skill-name">...</span>`，把里面的文字改成你的技能。想增加/减少技能就复制或删除一整块 `<div class="skill-card">...</div>`。 |

---

## 4. 工作经历

| 要改的内容 | 在 index.html 里搜什么 | 说明 |
|------------|------------------------|------|
| 每条工作的职位 | `exp-title` | 在 `<h3 class="exp-title">` 和 `</h3>` 之间改。 |
| 公司名和雇佣类型 | `exp-meta` | 在 `<p class="exp-meta">` 和 `</p>` 之间改，格式建议：**公司名 · 全职/兼职**。 |
| 时间范围 | `exp-date` | 在 `<span class="exp-date">` 里改日期文字，例如「2020年5月 - 2025年3月」。 |
| 工作内容（ bullets） | `exp-bullets` | 在 `<ul class="exp-bullets">` 里有很多 `<li>...</li>`，每条改一句；想增加就加一行 `<li>新内容</li>`。 |
| 技能标签 | `exp-tags` 里的 `tag` | 每个 `<span class="tag">...</span>` 是一个标签，改文字或增删整段 `<span class="tag">...</span>`。 |
| 整段工作经历 | `exp-card` | 一段工作 = 一个 `<article class="exp-card">...</article>`。要新增经历就复制一整段 `exp-card` 粘贴到「工作经历」的 `experience-list` 里；要删就删掉整段。 |

---

## 5. 实习经历

| 要改的内容 | 在 index.html 里搜什么 | 说明 |
|------------|------------------------|------|
| 职位、公司、时间、内容、标签 | 同「工作经历」 | 在 **实习经历** 那个 `section` 里找 `exp-card`，结构和「工作经历」完全一样，按上面工作经历的方式改。 |
| 增加/删除实习 | `id="internship"` | 在该 section 的 `experience-list` 里复制或删除整段 `exp-card`。 |

## 7. 关键成就 / 项目

| 要改的内容 | 在 index.html 里搜什么 | 说明 |
|------------|------------------------|------|
| 每条成就的标题 | `project-title` | 在 `<h3 class="project-title">` 和 `</h3>` 之间改。 |
| 描述 | `project-desc` | 在 `<p class="project-desc">` 和 `</p>` 之间改。 |
| 标签 | `project-tags` 里的 `tag tag-outline` | 改 `<span class="tag tag-outline">...</span>` 里的文字。 |
| 「了解更多」链接 | `project-link` | 把 `<a href="#">` 的 `#` 改成真实链接（没有可留 `#`）。 |
| 增加/删除一条成就 | `projects-grid` 里的 `project-card` | 每条 = 一个 `<article class="project-card ...">...</article>`，复制或删除整段；注意保留 `project-card--purple` / `--blue` / `--green` 其中一种样式。 |

## 9. 联系

| 要改的内容 | 在 index.html 里搜什么 | 说明 |
|------------|------------------------|------|
| 标题「一起合作」 | `一起合作` | 直接改。 |

| 「下载简历」按钮 | `下载简历` 和 `href="#"` | 把 `href="#"` 改成简历文件链接或下载地址。 |

## 11. 感谢页（thank-you.html）

打开 **`thank-you.html`**，可以改：

- 标题「感谢您的关注」
- 下面一段感谢文字
- 「返回首页」和「联系我」的链接已指向 `index.html` 和 `index.html#contact`，一般不用改。

---

## 小提示

- 修改前可先复制一份 `index.html` 做备份。
- 保存后若浏览器没变化，尝试 **Ctrl + F5** 强制刷新。
- 不要删掉类似 `class="..."` 这样的属性，只改标签之间的**文字**和 `href="..."`、`mailto:...` 里的链接即可。

网页做好并确认内容无误后，再按 README 里的步骤部署到 GitHub Pages 即可。
